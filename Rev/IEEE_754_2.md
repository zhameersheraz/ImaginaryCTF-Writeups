## Background Knowledge (Read This First!)

### What is Sigbovik?

**SIGBOVIK** is a satirical computer science conference ("Special Interest Group on Harry Bovik") that publishes joke papers. The referenced paper is by Tom7: **"Fluint8: Float-simulated Unsigned Integers"**.

### What is the Core Idea?

Tom7's paper demonstrates that you can **simulate integer arithmetic using only floating-point operations**. Specifically, IEEE 754 floating-point arithmetic (addition, multiplication, etc.) can be made to behave identically to integer arithmetic by exploiting the exact bit-level representation of floats.

This means a program that looks like it uses only floating-point math is actually performing hidden integer computations — making reverse engineering very confusing.
